This little webpage allows the user to make llm requests with an old computer (and an old browser).

It was only tested on an iMac G3 using the last version availible of interwebPPC.

It's really easy to use :

1) Put the index.html file on your old computer (USB stick, Floppy, CD, whatever !)

2) Get an API from https://openrouter.ai/

3) Put your API in the dedicated field

4) Reload the model list
